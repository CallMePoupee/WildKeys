# WildKeys

by [Charlotte Chapdelaine](https://github.com/CallMePoupee) ([@CallMePoupee](https://github.com/CallMePoupee))

Windows desktop app that pastes a **random line** from your text lists with a global hotkey — a simple wildcard substitute for AI tools and any other app that accepts keyboard input.

## Why WildKeys?

Many AI tools and websites do not support wildcard syntax. WildKeys provides a simple alternative by inserting a randomly selected entry from one of your wildcard lists with a single hotkey.

Because WildKeys works independently of the target application, it can be used anywhere keyboard input is accepted. This makes it compatible with virtually any software, including AI websites, desktop applications, browsers, and automation tools.

## How It Works

1. Create one or more text files containing your wildcard entries, with one entry per line.
2. Assign each file to a global hotkey.
3. Place the cursor anywhere text can be entered.
4. Press the assigned hotkey.
5. WildKeys inserts a randomly selected line from the associated text file.

## Installation

1. Download the latest release from the **Releases** page.
2. Run `WildKeys-Setup.exe` (or `WildKeys.exe` from the portable build).
3. Configure your wildcard lists and hotkeys in the app.

## Requirements

* Windows 10 or later
* Microsoft Edge WebView2 (usually preinstalled on modern Windows)
